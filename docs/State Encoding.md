# State Encoding 

State Encoded using binary values for implementation using D Flip Flops

### Example:

## State Encoding

| State | Encoding |
|--------|----------|
| S0 | 000 |
| S1 | 001 |
| S2 | 010 |
| S3 | 011 |
| S4 | 100 |

The binary encoding serves as the physical representation of each state within the hardware.
